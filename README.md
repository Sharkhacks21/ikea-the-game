# Bla-hajimemashite
Nice to meat you!

## About Bla-hajimemashite
“Hajimemashite” is used in Japanese the same way “Nice to meet you” is used in English
The tagline is a play on the word meet, referencing Ikea's famous Swedish Meatballs
